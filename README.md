# AWS EC2 Static Website Hosting

## Description
This project demonstrates how to host a static website on an AWS EC2 instance using Apache web server.

## Services Used
- AWS EC2
- Apache HTTP Server

## Steps Followed
1. Created an EC2 instance using Amazon Linux 2
2. Installed and started Apache web server
3. Deployed a static HTML website
4. Accessed the website via public IP

## Output
Website successfully hosted and accessible through EC2 public IP.
